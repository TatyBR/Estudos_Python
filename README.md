
# 📚REPOSITÓRIO DE ESTUDOS **PYTHON** 🐍

## 🔍 Visão Geral
Este repositório tem como objetivo reunir alguns exercícios práticos e projetos que estão sendo realizados no decorrer dos meus estudos sobre **Python**.

## 🎯 Objetivo
Organizar e documentar minha jornada de aprendizado, mantendo exemplos práticos e exercícios resolvidos que possam servir como referência futura.



## 🧱 O que há até agora:
📂 Pasta principal `Estudos_Python/` composta pelas subpastas `Exercicios/` e `Projetos_Iniciantes/`, onde:

🚀 `Exercicios/` possue vários exercícios que englobam alguns conceitos básicos:
- `1.Conceitos_basicos` → Scripts abordandos conceitos básicos como print, criação de variáveis, cálculos com variáveis e condicionais.
- `2.Calculadora` → Script que simula uma calculadora com Soma, Subtração, Multiplicação, Divisão, Potência, Raiz Quadrada e Logaritmo Natural .
- `3.Adivinhação com While` → Joguinho de adivinhação de número utilizando a estrutura de repetição while.
- `4.Jogo de corrida POO/` → Joguinho que simula uma corrida. Criação de classes e objetos para fixação dos conceitos de POO.
- `5.Business Case 1 - Ficha de Academia/` → Geração de uma Ficha de Academia como um arquivo pdf com dados digitados pelo usuário.
- `6.Business Case 2 - Gerador de Arquivo Excel/` → Geração de um arquivo exce com dados do Yahoo Finance utilizando a biblioteca yfinance. 
- `7.Organizador de Arquivos/` → Script que organiza os arquivos dentro de uma pasta de acordo com o seu tipo. Fixação dos conceitos do módulo os do Python.
- `8.Analise Vendas/` → Criação de um DataFrame a partir da exportação de um arquivo csv e realização de tratamento dos dados, cálculos estatísticos e agregações e em alguma colunas.
- `9.Visualizando Dados Gráficos/` → Conhecendo e testando vários tipos de gráficos utilizando como base os datasets da biblioteca Seaborn.

🚀 `Projetos_Iniciantes/` com:
